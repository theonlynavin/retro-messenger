# retro-messenger

A simple terminal-based messenger with a retro look made in lieu of an EE5150 assignment.

---

## Features

- A terminal UI built with curses (and a good amount of prompting)
- HTTPS communication over TLS
- Encrypted messaging using symmetric cryptography
- Most features of a typical messenger application

---

## Requirements

Python 3.9 or higher.

## How to Run

Clone the repository, install the requirements and open the main file. It is that simple.

```bash
git clone <repository-url>  
cd retro-messenger  
pip install -r requirements.txt  
python main.py  
```
